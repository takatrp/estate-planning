# 資産承継 面談ナビゲーター

相続・贈与の相談時に、職員が操作しながらクライアントと同じ画面を見て整理するための面談支援ツールです。  
家族構成・資産構成・過去贈与・生命保険・配偶者取得割合を整理し、概算税額・リスク・打ち手・ビフォーアフター・根拠リンクを確認することを目的としています。

## GitHub Pagesでの公開

1. このフォルダ内のファイルをリポジトリへ配置
2. GitHub の `Settings` → `Pages`
3. `Deploy from a branch` を選択
4. `main` ブランチの `/root` または `/docs` を指定
5. 発行されたURLへアクセス

外部ライブラリは使っていないため、静的ホスティングで動作します。

## ファイル構成

- `index.html`：画面構成
- `styles.css`：デザイン
- `app.js`：入力、計算、表示ロジック
- `tax-rules.js`：税制マスタ、税率表、根拠リンク

## 第1版でできること

- 家族構成から法定相続人・基礎控除を概算
- 資産構成から正味財産、課税遺産総額、相続税総額を概算
- 死亡保険金の非課税枠を確認
- 暦年贈与、相続時精算課税、住宅取得資金贈与の概算確認
- 配偶者取得割合別に一次相続・二次相続の粗い比較
- 納税資金、分割困難、二次相続、生前贈与加算などのリスク表示
- 根拠ページリンクの確認
- JSON保存・読込
- 面談メモ印刷

## 第1版であえて未実装のもの

- 小規模宅地等の特例の完全判定
- 路線価・倍率方式による土地評価
- 非上場株式評価
- 貸付不動産評価の詳細
- 遺留分侵害額請求の計算
- 事業承継税制の詳細判定
- 国際相続
- 信託・法人化・不動産組替えの高度比較

これらは、論点フラグとして扱い、実行前に個別確認する前提です。

## 運用上の注意

本ツールは概算把握・論点整理・面談品質の標準化を目的としたものです。  
申告・実行判断にあたっては、財産評価、特例適用要件、家族関係、遺産分割、過去贈与、契約関係、税制改正を個別に確認してください。

## 税制マスタ更新

税制改正時は、まず `tax-rules.js` を更新してください。  
各制度の根拠ページ、確認日、税率表、控除額を分離しています。
